使用帮助

🚀 快速开始

  1. 首次使用请配置高德地图 API Key
  2. 输入出发地和目的地（或从常用地址/历史记录选择）
  3. 选择交通方式
  4. 设置目标到达时间和提前提醒时长
  5. 点击"设置出发提醒"，保持页面打开即可

🔑 如何申请高德地图 Key？

  1. 打开 高德开放平台 并注册登录
  2. 进入"控制台" → "应用管理" → "我的应用"
  3. 点击"创建新应用"，填写应用名称
  4. 点击"添加 Key"，服务平台选择"Web端(JS API)"
  5. 复制生成的 Key 填入本页面即可

💡 建议在高德控制台配置域名白名单，防止 Key 被盗用

⭐ 常用地址

点击"+添加"按钮可以保存常用地址（如家、公司）。
使用方法：先点击出发地或目的地输入框，再点击常用地址标签即可快速填入。

⏰ 提醒说明

  • 设置提醒后请保持页面打开
  • 到点会全屏弹窗 + 响铃 + 震动（手机端）
  • 点击"我知道了"关闭提醒
  • 关闭页面后提醒会失效

📤 部署到 GitHub Pages

  1. 在 GitHub 创建一个新仓库
  2. 将 index.html 文件上传到仓库
  3. 进入仓库 Settings → Pages
  4. Source 选择 main 分支，保存
  5. 稍等几分钟，访问 https://你的用户名.github.io/仓库名/ 即可

💡 也可以直接修改 index.html 中的 AMAP_KEY 变量，把 Key 写死在代码里，这样别人打开就不用再输入 Key 了

💾 数据存储

所有数据（常用地址、历史记录、Key）都保存在浏览器本地 localStorage 中，不会上传到任何服务器。 清除浏览器数据或换设备后数据会丢失。
